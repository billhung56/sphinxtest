緣起
^^^^

此文件的目標是作為

More story here ...
^^^^^^^^^^^^^^^^^^^
此文件緣起於2017年的暑假，哥哥找我和另外幾位同學一起在母校 `成功國中 <http://www.cgjh.hcc.edu.tw/>`_ 舉辦為期五天的活動，教國中生寫Python程式，同時希望帶入些計算機科學中有趣的想法。但五天實在有限，我也是第一次嘗試這種教學工作，且參與的學生從升國一到升高一的都有，學習狀況參差不齊，最後不得已取消了許多預定內容。當時是不用課本的，因為教學目的著重於程式運行邏輯和語法，想說用Python官方簡潔易懂的的教程(tutorials)作為參考資料就夠了。況且網路上還有一堆教學或免費的書可以看，遇到問題也不難解決或尋求更詳細的解釋才是。但我沒想到的是，許多人並沒辦法順暢的閱讀英文，年紀較小的同學對於使用搜尋引擎找資料可能也不熟練，結果就是很多人等者問問題，然後我手忙腳亂。想起來真的不妥，還是得考慮同學的能力因材施教才有效率。於是萌生了建立此文件的念頭，希望能提供在國、高中階段學習程式語言、電腦科學的同好一些參考及指引。


Philosophy
^^^^^^^^^^
這邊想強調一下，建立這份文件的精神，與當初辦教學活動是一致的，是希望幫助那些對喜歡、欣賞電腦／程式設計的夥伴，而不是刻意去宣傳資訊領域有多棒，要培養計算思維，好讓小朋友贏在起跑點、在未來可以做軟體工程師這類充滿競爭動機的功利想法。學海無涯，各個領域都有其深奧的學問和價值，軟體產業對現今社會的影響確實深遠，但也無須過度放大。對於近期程式教育的趨勢及各種熱議，我只能說：「先別想了，來這逛逛吧。」
